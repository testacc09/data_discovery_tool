# Инструмент поиска данных

Инструмент для автоматизированного обнаружения и семантического поиска метаданных таблиц и колонок для ИИ-агентов.

## 🚀 Быстрый старт

### 1. Установка окружения

Склонируйте репозиторий и выполните следующие команды в корневой директории проекта:

```bash
python -m venv venv
call venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Подготовка данных

Перед запуском поискового движка необходимо инициализировать базу данных и проиндексировать источники:

```bash
python scripts/seed_data.py
```

### 3. Запуск пользовательского интерфейса

Для работы с визуальным интерфейсом Streamlit:

```bash
streamlit run ui/app.py
```

## 🧪 Тестирование

Для проверки работоспособности всех модулей выполните unit-тесты:

```bash
pytest tests/
```

## 🔌 Интеграция с MCP

```bash
npx @modelcontextprotocol/inspector python mcp_server/server.py
```

## 📚 Документация проекта

* **ARCHITECTURE.md** — Внутреннее устройство и стек технологий.

* **MCP.md** — Спецификация инструментов для ИИ-агентов.

* **Evaluation.md** — Результаты тестирования и метрики качества.

Проект разработан для упрощения взаимодействия ИИ с распределенными источниками данных (SQLite, CSV, API).
